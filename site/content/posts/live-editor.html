Shortcuts

  • ctrl-space (mac) triggers autocomplete
  • cmd-i (mac) shows autocomplete
  • cmd-k cmd-i (mac) shows hover (also see Show Hover in the command palette)
  • F1 shows the command palette

  • Delete the resources: section
  • Begin typing resou and type ctrl-space
  • Select resource from the auto-complete
